Give it an id of malsRunningTotal Total: $0.00 6) Ensure your wwX and userid are the same as Mals (TIP! enter your details at the top before copying example) 7) Open the web page in a browser e.g. Chrome and test it --- Calling Script --- The script is very versatile. You can activate it on one or more forms (or links) easily. Note: if you don't know jQuery selectors visit https://www.w3schools.com/jquery/jquery_selectors.asp - Example 1 = specific form by id MalsScripts.priceOptions('#form1'); This will activate the script on the form with id="form1" - Example 2 = cart delimiter. The script uses {br} as the cart delimiter which spaces the product in the cart. You can change this to whatever you wish MalsScripts.priceOptions('#form1',','); - Example 3 = submit callback function MalsScripts.priceOptions('#form1','{br}',function(){ // Called when form is submitted }); - Example 4 = running total update callback priceOptions(element,,function(){ // Called when running total display is updated }) - Example 5 = callback called when input element is clicked/updated priceOptions(element,{br},,,function(){ // Called when input element is changed }) --- General Notes --- - Add class .mals-ignore to any forms you don't want transformed e.g.
