For example Let's break down the mals-img information mals-img="{'src':'card-red.png','target':'#malsImageDisplay'}" src = path/uri of image e.g. card-red.png target = id of image you want to change e.g. #malsImageDisplay1 (Note: use jQuery selector i.e. #ImageName) TIP! - Don't use spaces or strange characters in your image names - All lower case - Use a hypen (-) to separate image parts e.g. product-size-color.png - Use CSS to position images exactly where you want e.g. beside will overlap 1st then the next will overlap that - Position and scale the overlay images using the details below > Position: place the overlay exactly where you want. For example mals-img="{'src':'star.png','target':'#malsImageDisplay','overlay':'true','left':'39%','top':'15%'}" overlay = true left = xx top = yy TIP! - You can use %,px(pixels) or other values to position image - Play around with the values to position exactly - Combine with scale (see below) to reduce size of image > Scale: Reduce/Increase the size of the overlay image. For example mals-img="{'src':'star.png','target':'#malsImageDisplay','overlay':'true','scale':'40%'}" scale = xx% Notes: i) Written in json format which is quite strict and the format 'name':'value' ii) " and ' count.
